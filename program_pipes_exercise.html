Function composition

Click for the solution 

summarize(
  group_by(
    filter(gun_deaths, sex == "M", race == "White", year == 2012),
    education),
  n = n()
)

Piped operation

Click for the solution 

gun_deaths %>%
  filter(sex == "M", race == "White", year == 2012) %>%
  group_by(education) %>%
  summarize(n = n())
## # A tibble: 5 x 2
##      education     n
##         <fctr> <int>
## 1 Less than HS  2858
## 2       HS/GED  7912
## 3 Some college  4258
## 4          BA+  3029
## 5           NA   285
